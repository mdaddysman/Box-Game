# SDL Learning

An example git for learning SDL 2.0 with Visual C++ 2013

Requires in c:\sdl2: 
- SDL2
- SDL_Mixer

The following dll's must be copied to the Debug folder:
- `SDL2.dll`
- `SDL2_mixer.dll`
- `smpeg2.dll`

Resources [download](https://www.dropbox.com/s/gmjd1hczqn4xr1k/experiment.zip?dl=1): 
- `Music/Game1.mp3`
- `Sounds/fail.wav`
